# Student Management System

A **console-based C program** to manage student records. You can **add, update, delete, search, display, save, and load** student information. Data is stored in memory and can be saved/loaded from `students.txt`.

---

## Features

- Add students with **ID, name, 3 grades, address, phone**  
- Update student details interactively  
- Delete students by ID  
- Search student by ID  
- Display all students in a neat table  
- Save and load data from a file  

---

## File Format (`students.txt`)

studentID|name|grade1|grade2|grade3|address|phone


Example:

101|Alice Johnson|85|92|78|123 Maple Street|555-1234
